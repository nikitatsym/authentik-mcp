Authentik MCP

Manage users, groups, applications, flows, and policies on Authentik from Claude, Cursor, or any MCP client.


  


  

    
Where to paste

    
Claude Desktop — ~/Library/Application Support/Claude/claude_desktop_config.json

    
Cursor — .cursor/mcp.json in your project

    
Claude Code — ~/.claude.json top-level mcpServers

  


  

    
Getting an API token

    
Go to your Authentik admin panel → Directory → Tokens and App passwords → Create a token with the API scope.
